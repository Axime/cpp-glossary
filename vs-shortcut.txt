

F1/F2: 列出成员(快速提示)
F4: 转到定义(快速查找引用)
F5: 调试，逐语句(单步跳入)
F6: 调试，逐过程(单步跳过)
F8: 停止调试

F11 / Ctrl + Enter: 开始调试
Ctrl + F11: 开始运行(不调试)

Ctrl + (Shift) + Q: 添加/移除注释
Tab + (Shift): 缩进 / 缩回

Ctrl + A: 全选
Ctrl + S: 保存
Ctrl + D: 快速复制当前行
Ctrl + Shift + D: 快速删除当前行
Ctrl + (shift) + F: 查找(全局)
Ctrl + (shift) + H: 替换(全局)

Ctrl + Z: 撤销
Ctrl + X: 剪切
Ctrl + C: 复制
Ctrl + V: 粘贴

: 调试启动
Alt + Enter: 快速查看项目属性

Ctrl + B: 构建/生成项目
